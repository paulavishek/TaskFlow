## 🚀 Quick Start (15 Minutes)

### 1. Get Free API Key
- Go to: https://ai.google.dev
- Click: "Get API Key"
- Copy the key

### 2. Create `.env` File
```bash
GEMINI_API_KEY=your_key_here
```

### 3. Run Migrations
```bash
python manage.py migrate
```

### 4. Start Server
```bash
python manage.py runserver
```

### 5. Open Browser
```
http://localhost:8000/assistant/
```

**Done!** You're live. 🎉
